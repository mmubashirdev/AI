# Jupyter Notebooks

This directory contains interactive Jupyter notebooks for tutorials and experiments.

## Contents

Notebooks are organized by topic:
- `machine-learning/` - ML tutorials and examples
- `deep-learning/` - Neural network tutorials
- `computer-vision/` - CV projects and demos
- `reinforcement-learning/` - RL experiments
- `generative-ai/` - Generative model tutorials

## Running Notebooks

1. Install Jupyter:
```bash
pip install jupyter notebook
```

2. Start Jupyter server:
```bash
jupyter notebook
```

3. Navigate to the desired notebook and open it

## Notebook Guidelines

- Clear all outputs before committing
- Include markdown cells with explanations
- Keep notebooks focused on a single topic
- Add requirements at the top of each notebook
- Include visualization of results
